# CoLAB — Regulatory Intelligence

Global pharmaceutical regulatory submission intelligence platform.

- 128+ countries · CTD + ACTD support
- Full ASEAN bloc (10 member states) with native ACTD
- ICH M4Q(R2) R1/R2 toggle
- AI Document Review (40+ document types)
- Pharmacovigilance module
- eCTD export · LORENZ compatible
